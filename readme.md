# Gistologia Bot
🚀 Бот для изучения гистологии

## Что из себя представляет бот?
```
✍️ Доступно 9 разделов для изучения

📅 Пробная подписка на 1 день

👨‍🦲 Добавление пользователей в полный доступ
```

## ✅ Инструкция 

1. Региструемя на [Render](https://dashboard.render.com/)
2. Нажимаем New - Web Service
3. Ищем через ссылку репозиторий
4. Start Command: **python main.py**
5. Добавляем переменные в Environment Variables

По вопросам в [тг](https://t.me/hundrik3)
